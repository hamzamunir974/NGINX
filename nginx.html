How to Install Nginx Server

What is Nginx?

Nginx is an open-source web server that can also be used as a reverse proxy, load balancer, and HTTP cache. It's known for its high performance and stability, and it's widely used to serve static content on websites.

Prerequisites

  • A server running Linux (Ubuntu, CentOS, etc.)
  • Access to the command line
  • Root or user with sudo privileges

Steps to Install Nginx

  • Step 1: Update Your Package List

    Before installing Nginx, ensure your package list is up to date. Run the following command:

    sudo apt update

  • Step 2: Install Nginx

    Install Nginx using the package manager by running:

    sudo apt install nginx

  • Step 3: Start and Enable Nginx

    Once installed, start the Nginx service:

    sudo systemctl start nginx

    Enable it to start automatically on boot:

    sudo systemctl enable nginx

  • Step 4: Adjust Firewall Settings

    If you have a firewall running, allow traffic on HTTP (port 80) and HTTPS (port 443):

    sudo ufw allow 'Nginx Full'
